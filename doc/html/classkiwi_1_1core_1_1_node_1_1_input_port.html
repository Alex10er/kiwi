Detailed Description

template<class SlotType>
class kiwi::core::Node::InputPort< SlotType >

Generic input port class for Reader and Writer interface.

An instance of this class is hold by a kNode for each of it's inputs. Port classes are designed to do most of the external actions on Node/Filter which are doning connections between Nodes and retrieving informations on their input/output data.

Each port has a name which use is facultative has they are truely accessed using an integer index.
