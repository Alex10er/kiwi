Detailed Description

template<typename TValueType, unsigned int TDimension>
class kiwi::generic::ArrayWriter< TValueType, TDimension >

Writer class able to read and write data in any Container inheriting from kiwi::generic::AbstractArrayContainer.

(like kiwi::generic::ArrayContainer, kiwi::generic::MultiArrayContainer or kiwi::image::CairoImageContainer).
