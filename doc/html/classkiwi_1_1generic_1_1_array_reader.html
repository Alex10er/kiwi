Detailed Description

template<typename TValueType, unsigned int TDimension>
class kiwi::generic::ArrayReader< TValueType, TDimension >

Reader class able to read data from any Container inheriting from kiwi::generic::AbstractArrayContainer.

(like kiwi::generic::ArrayContainer, kiwi::generic::MultiArrayContainer or kiwi::image::CairoImageContainer).
