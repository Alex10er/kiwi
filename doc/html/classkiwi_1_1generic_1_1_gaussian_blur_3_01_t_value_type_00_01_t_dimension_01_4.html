Detailed Description

template<typename TValueType, unsigned TDimension>
class kiwi::generic::GaussianBlur< TValueType, TDimension >

Generic gaussian blur Filter.

This Filter can be applied to any array based Container of dimension superior to zero and with a scalar type that has basic arithmetic operators (+,-,/...).

Input ports

Reader

The first Reader input port corresponds to the blur strength:

  • if a Value_uint32 is connected, the blur will be applied uniformly.
  • if an ArrayNd_uint8 is connected (with N equal to TDimension), this input will be used as a mask that regulates the blur radius for each fragment.
    The next K Reader input ports are the image inputs that we want to blur. They must be of any type compatible with ArrayReader<TValueType, TDimension> (Container class deriving from AbstractArrayContainer). The number K of these ports is not limited.
