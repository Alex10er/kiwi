Detailed Description

template<typename ValueTypeT>
class kiwi::generic::ArrayConstIterator< ValueTypeT >

const iterator class working with contiguous data containers.

This iterator class has been designed to work with ArrayContainer but is compatible with any continuous data container. The iterator needs to know the starting pointer and the pointer to the last value, as well as the increment value to iterate through interleaved arrays.
