virtual bool kiwi::core::Filter::readyCondition (  )  [inline, protected, virtual]

a method to overload when there is need to test things when checking readiness

This method may optonally be overloaded in some case when it is convenient to add an extra condition for checking wether the Filter is ready to compute. This is called within isReady() and does not replace setting the ready state with void setReady(bool), isReady checks that boths conditions are fulfilled.

by default returns true so that it doesn't interfere with setReady(bool) when the method is not overloaded

void kiwi::core::Filter::setReady ( bool  state  )  [inline, protected]

set a state of the Filter.

Partially set the "readiness" state of this Filter. Note that if readyCondition() is overloaded, both readyCondition and the readiness state must be true for isReady() to return true.
